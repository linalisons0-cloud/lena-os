# Lena OS 6.1 Rebuilt

Новая React-версия приложения, основанная на функциональности Lena OS 6.1.

## Уже работает
- экран «Главная» и ежедневные привычки;
- Glow Up Score;
- чтение с минутами и историей;
- расчёт КБЖУ;
- дневник питания и калораж за сегодня;
- журнал тренировок;
- дневник кожи;
- замеры веса, талии и ягодиц;
- журнал самочувствия и настроения;
- локальный помощник Lena AI;
- локальное сохранение данных;
- адаптивный дизайн для iPhone.

## Netlify
- Branch: `main`
- Build command: `npm run build`
- Publish directory: `dist`
- Base directory: оставить пустым

## Важно
Данные хранятся локально в браузере устройства.
