LENA OS 9.6 — STABLE SAVE

Исправлено:
- бюджет, доходы, расходы, вода и остальные данные сохраняются отдельно от фотографий;
- фотографии «до/после» хранятся в IndexedDB и больше не переполняют localStorage;
- повреждённые старые данные не ломают запуск;
- service worker обновляет страницу, а не держит старую версию в кэше.

Для GitHub Pages загрузите распакованные файлы с заменой старых. ZIP в репозиторий загружать не нужно.
